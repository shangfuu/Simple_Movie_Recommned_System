## Hybrid Recommend
* 結合”相似電影”和”使用者喜好”來推薦
* 使用第一個方法，可以使用卡斯、導演、關鍵詞等，計算模擬分數，找出推薦的電影。
* 使用上述得到結果，再利用使用者喜好，即可推算出特定使用者的電影喜好
